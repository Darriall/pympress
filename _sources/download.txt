Download pympress
=================

Stable releases
---------------

The latest stable release is **pympress 0.3**, available as a `tarball`_ or as a
`zipball`_.

You can download older releases from the `GitHub download page`_.


Development version
-------------------

pympress source code is hosted on `GitHub`_. You can browse the code using the
`web interface`_, or you can download it using git: ``git clone
git://github.com/Schnouki/pympress.git``

.. _zipball: https://github.com/Schnouki/pympress/zipball/v0.3
.. _GitHub download page: https://github.com/Schnouki/pympress/downloads
.. _GitHub: https://www.github.com/
.. _web interface: https://github.com/Schnouki/pympress/tree/master
